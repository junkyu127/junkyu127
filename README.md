### 💻 핵심 실무 역량

#### 1️⃣ 이커머스 API 통합 및 데이터 파이프라인 구축
* **API 연동 자동화:** 국내외 주요 쇼핑몰 플랫폼의 API를 분석하여 주문 수집부터 송장 전송까지 전 과정 자동화 프로세스 구축.
* **데이터 표준화(Normalization):** 플랫폼별 상이한 API 규격을 내부 공통 모델로 통합하여 데이터 정합성 유지 및 시스템 확장성 확보.
* **예외 처리 및 안정성:** 호출 제한(Rate Limit) 대응 재시도 로직과 다중 검증 시스템을 통해 데이터 유실 및 운영 사고 원천 차단.

#### 2️⃣ 리눅스 기반 인프라 운영 및 풀스택 유지보수
* **서버 관리 및 모니터링:** SecureCRT를 활용해 리눅스 환경에 접속하여 실시간 로그 분석, 서버 자원 점검 및 장애 대응 수행.
* **엔드투엔드(End-to-End) 개발:** 서버 엔진 설정부터 비즈니스 로직 구현, 사용자 화면 디자인 반영까지 서비스 전체 레이어 유지보수.

#### 3️⃣ 맞춤형 비즈니스 로직 설계 및 최적화
* **전략적 로직 구조화:** 고객사별 개별 요구사항(전용 기능 버튼, 특수 로직 등)을 대응하기 위해 하드코딩된 코드를 분리된 레이어로 재구성하여 유지보수성 향상.
* **시스템 최적화:** 대용량 주문 데이터 처리를 위한 DB 인덱스 최적화 및 서버 부하 분산 처리.

### 💡 주요 문제 해결 사례 (Troubleshooting)

* **상품 단위 독립 배송 로직 구현**
  * 단일 주문 내 복수 상품의 개별 배송 처리를 위해 DB 스키마를 보완하고, API 파라미터 매핑 로직을 정교화하여 비즈니스 유연성 확보.
* **IP 제한 및 보안 네트워크 환경에서의 API 연동 해결**
  * **문제:** 쇼핑몰 API의 IP 화이트리스트 제한으로 인해 특정 환경에서 직접 호출이 불가능한 상황 발생.
  * **해결:** C# 클라이언트와 외부 API 사이에서 프록시 역할을 수행하는 PHP 가공 레이어 구축. `[C#] -> [PHP 프록시] -> [쇼핑몰 API]` 구조로 데이터를 가공하여 JSON 형태로 전달.
* **신규 연동 리스크 최소화**
  * 신규 API 도입 시 테스트베드(Test-bed) 환경 구축 및 분기 처리 레이어 격리를 통해 운영 환경의 배포 안정성 제고.
